# Atividade 3 - Lógica

## Descrição
Este projeto contém exercícios de lógica de programação e desenvolvimento front-end, aplicando conceitos fundamentais de JavaScript e HTML/CSS.

## Como usar
1. Clone ou baixe este repositório
2. Abra os arquivos HTML no seu navegador
3. Execute o jogo de advinhação.

## Objetivo
Praticar e consolidar conhecimentos em lógica de programação, estruturas de controle e manipulação do DOM.
